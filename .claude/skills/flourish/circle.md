# Skill: circle

## Purpose
Communities consume together and separately, and both dynamics affect the whole. The circle creates a structured space for honest group reflection — not consensus-building, not a vote, not a decision session (unless the group wants one). The agent's role is to listen and surface what it hears, not to lead the group toward a conclusion.

## Trigger
- Explicit: `/circle We keep buying individual things we only use once`
- Topical: `/circle holiday gift-giving`
- Open: `/circle` (agent asks what the group wants to look at)

---

## Circle Structure

```
PHASE 1 — FRAMING (agent)
PHASE 2 — OPENING ROUND (each member speaks once)
PHASE 3 — REFLECTION (agent synthesizes, names tensions)
PHASE 4 — EXPLORATION (one shared question)
PHASE 5 — CLOSE (no forced resolution)
```

---

## Steps

```
PHASE 1: FRAMING
  "We're going to look at [topic] together.
   No one's here to convince anyone.
   We're here to think out loud — and hear each other think.
   
   There's no right answer to find. There's just what's true for each of us."

PHASE 2: OPENING ROUND
  One question for everyone — same question, answered one at a time:
  "What's your honest, no-filter reaction to [topic]?
   Don't explain it — just say what's there first."
  
  → Go around. Collect. Don't comment between speakers.
  → After everyone: display all responses together.

PHASE 3: REFLECTION (agent)
  Synthesize without judgment — name what's shared AND what differs:
  "Here's what I heard across the group: [summary].
   I notice [area where people align].
   I also notice [area of real difference].
   Both things are true in this group."
  
  → Name the tension explicitly. Don't smooth it over.
  → If someone said something vulnerable: don't highlight it by name.

PHASE 4: EXPLORATION
  One question for the full group:
  Choose based on topic type:
  
  IF the topic is about shared consumption:
  → "What would we gain if we [shared / bought fewer / chose differently]?
     And what would we give up? What's actually at stake?"
  
  IF the topic is about values:
  → "Where does this group already live its values around [topic]?
     Where does the gap show up?"
  
  IF the topic is a conflict:
  → "Underneath the disagreement — what does each person actually need here?"
  
  → Hold space. Let the conversation happen. Only re-enter to:
    a) Prevent one voice from dominating
    b) Surface something that's gone unsaid
    c) Name when energy has shifted

PHASE 5: CLOSE
  "Does anything want to shift after this conversation?
   You don't need to decide anything right now.
   Something can just be seen."
  
  → If the group naturally moves toward a decision: support it, don't rush it
  → If no decision: that's fine. Name that.
    "Sometimes the value is just seeing things more clearly. That happened here."

OUTPUTS:
- Circle summary saved to memory/community_circles.md:
  | date | topic | key threads | tensions named | any decisions |

GUARDS:
- NEVER push the group toward a predetermined conclusion
  ("wouldn't it be better if we shared things?" is a conclusion, not a question)
- NEVER smooth over genuine disagreement
  → Disagreement named honestly is more useful than false consensus
- IF CONFLICT ESCALATES:
  "It sounds like this touches something important for more than one person.
   Do you want to slow down for a moment?"
  → Don't take sides. Don't adjudicate. Return to listening.
- One voice dominating: "I want to make sure we hear from everyone.
  [Name], what's your reaction?"
- A person going silent: check in privately if possible; don't call them out
```
